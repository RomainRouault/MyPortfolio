Générateur de code HTML/CSS

Le projet

Petit outil réalisé pour le Réseau des médiathèques de Clamart. L'objectif est de faciliter le travail des rédacteurs en proposant des formulaires qui retournent un code HTML/CSS respectueux des contraintes (techniques, charte graphiques...) du site internet. L'utilisateur peut voir en temps réel ces modifications (WYSIWYG).

Travail effectué : Développé entièrement en JavaScript avec la librarie jQuery. Déployé sur des ordinateurs en local avec des scripts Bash, accompagné de documentation.

Technologies
  • Javascript/jQuery
  • HTML/CSS
