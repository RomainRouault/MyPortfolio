BiB92

Le projet

Projet initié en concertation avec l'association des bibliothèques des Hauts-de-Seine BiB92. L'objectif est de refondre intégralement un site obsolète et vulnérable en terme de sécurité tout en conservant les données.

Travail effectué : Conception et développement d'un template responsive pour le CMS Joomla. Conseils, migration des données, graphisme, intégration, déploiement et maintenance.

Technologies
  • Joomla 3
  • HTML/CSS
  • Bootstrap 3
  • Less
  • PHP/MySQL
  • jQuery/jQuery UI
  • Twitter API
