Newsletter - Médiathèques CCA

Le projet

Un template de newsletter réalisé pour le réseau des médiathèques de CCA, service de Concarneau Cornouaille Agglomération.

Travail effectué : Conception et développement d'un template responsive avec le framework Foundation for Emails dans le respect de la charte graphique préexistante.

Technologies
  • Foundation for Emails 2
  • HTML/CSS
  • Saas
